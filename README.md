# How to start?

### Pre-requirements
- MongoDB
- Node
- npm / yarn / pnpm

### Instructions:
1) Install dependencies: 
```bash
yarn install
```
```bash
npm install
```
```bash
pnpm install
```

2) Execute the following command:
```bash
npm run dev
```

3) Enter the localhost link with port `:3000`, you can change it on `src/server/config.js`

4) Enjoy!
